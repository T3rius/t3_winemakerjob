# t3_winemakerjob

For support: [Discord](https://discord.gg/yZv3F6Bhd6)

Features:
* Easy to configurate
* Boss Menu
* Garage
* No AFK Job
* Job Blips
* More types of wine
* Webhooks
* Harvesting

Dependencies:
* [ox_lib](https://github.com/overextended/ox_lib)
* [ox_target](https://github.com/overextended/ox_target)
